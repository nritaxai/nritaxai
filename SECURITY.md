# Security Policy

## Reporting a vulnerability

Please report suspected security issues privately to `security@nritax.ai` or `support@nritax.ai`.

Include:

- affected endpoint, page, or workflow
- reproduction steps
- impact assessment
- any proof-of-concept or logs with sensitive data redacted

Please do not post unpatched vulnerabilities publicly.

## Scope

This repository contains application, payment, observability, and security controls for `nritax.ai`.

## Response expectations

- Initial acknowledgement target: 3 business days
- Triage target: 7 business days
- Critical issue mitigation target: as fast as operationally possible, with rollback-first handling for production safety
